# 过马路（经典版）

一个纯 HTML/CSS/JavaScript 的小型过马路游戏。

## 新内容

- 四档难度：简单、困难、极难、炼狱
- 更换角色材质
- 增加车辆种类
- 更换车辆材质
- 狙击手随机现身、追踪与锁定机制
- 狙击击杀 GIF 特效
- 被击败音效（用户提供 MP3）
- 通关计时器
- 胜利/失败屏幕中央 3D 大字提示
- 失败时按死亡来源显示副标题
- 手机版触控按钮与移动端适配（桌面端界面保持不变）
- 右上角更新日志
- 右下角版本号显示

## 玩法

- 方向键或 WASD 移动
- 躲开车辆，一直往上走
- 还要躲开会在开局 5 秒内随机出现并追踪你的狙击准星，被锁定后会开枪
- `R` 重新开始

## 运行

直接双击打开 `index.html` 即可运行。

也可以在本地静态服务器中打开。

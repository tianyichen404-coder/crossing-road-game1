# 过马路（经典版）

一个纯 HTML/CSS/JavaScript 的小型过马路游戏。

## 新内容

- 四档难度：简单、困难、极难、炼狱
- 更换角色材质
- 增加车辆种类
- 更换车辆材质
- 狙击手随机现身、追踪与锁定机制
- 狙击击杀 GIF 特效
- 被击败音效（用户提供 MP3）
- 通关计时器
- 胜利/失败屏幕中央 3D 大字提示
- 失败时按死亡来源显示副标题
- 手机版触控按钮与移动端适配（桌面端界面保持不变）
- 无尽模式开场倒数结束前禁止使用 W 往前移动
- 仅无尽模式暂时移除狙击手
- 右上角更新日志
- 右下角版本号显示
- 无尽模式 5 秒开场倒数与连续流水线地图生成
- 玩家开局/复活的 3.1 秒无敌机制当前不生效
- 新增 P 键暂停 / 继续，移动端也有暂停按钮
- 无尽模式新增默认 BGM 与音乐开关
- 新增角色选择区：坤坤 / 牢大
- 无尽模式 BGM 改为倒数结束后开始循环播放
- 角色在游戏内保持原图比例缩放，不再被拉伸变形

## 玩法

- 方向键或 WASD 移动
- `P` 暂停 / 继续，`R` 重新开始
- 游戏开始前可在角色选择区选择坤坤或牢大
- 无尽模式倒数结束后默认播放 BGM，可用“音乐”按钮开关
- 躲开车辆，一直往上走
- 还要躲开会在开局 5 秒内随机出现并追踪你的狙击准星，被锁定后会开枪

## 运行

直接双击打开 `index.html` 即可运行。

也可以在本地静态服务器中打开。
